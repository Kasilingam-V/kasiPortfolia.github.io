Reference & Recreation

This project is a recreation of interaction systems seen in modern first-person puzzle games. The focus was to reverse engineer and rebuild the mechanics independently using Unreal Engine.

Challenges & Solutions

Issue

Previously interacted objects remained highlighted even after the player changed focus.

Solution

Implemented a reference tracking system for the last hit actor and ensured it is cleared before applying new interaction logic. This guarantees only one active object at a time.

Issue

Runtime errors occurred when destroying actors during interaction.

Solution

Added validity checks before executing destroy operations, ensuring safe execution and preventing invalid memory access.

Player Experience Impact

The interaction system provides clear, responsive feedback, allowing players to confidently identify and manipulate objects. This reduces confusion and ensures smooth gameplay flow in puzzle scenarios.
